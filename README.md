# Presentación - Tercer mes

Proyecto pequeño en React + Vite + Tailwind + Framer Motion para crear una presentación romántica por el tercer mes.

Requisitos:
- Node.js >= 16

Instalación:

```bash
npm install
npm run dev
```

Uso:
- Coloca las imágenes de la pareja dentro de la carpeta `img/` en la raíz del proyecto. El app las importa automáticamente.
- Navega con los botones "Anterior" y "Siguiente".
